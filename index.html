How to Use

  1. Search for your game using the search bar above
  2. Click download - you'll get a .lua file or .zip if multiple files needed
  3. Extract (if ZIP) and move all .lua files to SteamTools/Manifests
  4. Restart Steam and enjoy
Rockstar Games: These require additional launcher manifests which are automatically included in the ZIP download.
